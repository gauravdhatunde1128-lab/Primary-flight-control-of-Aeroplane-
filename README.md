# Primary-flight-control-of-Aeroplane-
Primary Flight Controls ✈️
Primary flight controls are the main controls used to directly control the aircraft’s attitude and direction in flight.
Without these, the airplane cannot be flown properly.
Types of Primary Flight Controls
There are three primary flight controls:
1. Ailerons
Control roll (banking left or right)
Located on the outer trailing edge of the wings
Used for turning the aircraft
2. Elevator
Controls pitch (nose up or nose down)
Located on the horizontal tail (stabilizer)
Used to climb or descend
3. Rudder
Controls yaw (nose left or right)
Located on the vertical tail (fin)
Used to maintain directional stability
